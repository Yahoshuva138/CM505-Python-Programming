# Python Lab Experiments

A comprehensive collection of Python programs covering fundamental concepts for 10th-class students.

## Programs Overview

### 1. Data Types Programs (01-03)
- **01_data_types_numbers.py** - Working with integers and floats
- **02_data_types_strings.py** - String operations and methods
- **03_data_types_compound.py** - Tuples, lists, and dictionaries

### 2. Arithmetic & Conversions (04-05)
- **04_currency_conversion.py** - Convert USD to Indian Rupees
- **05_data_size_conversion.py** - Convert bits to MB, GB, TB

### 3. Geometry Calculations (06-07)
- **06_square_area_perimeter.py** - Square calculations
- **07_cone_volume_area.py** - Cone volume and surface area

### 4. Conditional Logic (08-09)
- **08_odd_even_checker.py** - Determine odd or even numbers
- **09_greatest_of_three.py** - Find greatest of three numbers

### 5. Loops & Factorial (10-11)
- **10_factorial_loops.py** - Factorial using for and while loops
- **11_multiplication_table_loops.py** - Multiplication tables using loops

### 6. Functions (12-13)
- **12_factorial_functions.py** - Factorial using functions
- **13_multiplication_table_functions.py** - Multiplication tables using functions

### 7. Recursion (14-15)
- **14_factorial_recursion.py** - Factorial using recursion
- **15_fibonacci_recursion.py** - Fibonacci sequence using recursion

### 8. List Operations (16-17)
- **16_list_operations_basic.py** - Create, add, delete list elements
- **17_list_sort_reverse_count.py** - Sort, reverse, and count list elements

### 9. Dictionary Operations (18)
- **18_dictionary_operations.py** - Create, add, delete dictionary elements

### 10. Statistics (19)
- **19_statistics.py** - Calculate average, mean, median, standard deviation

### 11. Factors (20)
- **20_factors.py** - Find factors of a number

### 12. File I/O (21-22)
- **21_file_io_write.py** - Create files and write content
- **22_file_io_read_replace.py** - Read files and replace characters

### 13. Object-Oriented Programming (23-25)
- **23_complex_numbers_addition.py** - Add complex numbers using classes
- **24_complex_numbers_subtraction.py** - Subtract complex numbers using classes
- **25_packages_and_modules.py** - Python packages and modules

## How to Run Programs

Each program can be run independently using Python:

```bash
python 01_data_types_numbers.py
python 04_currency_conversion.py
python 23_complex_numbers_addition.py
```

## Concepts Covered

- **Data Types**: Numbers, strings, lists, tuples, dictionaries
- **Operators**: Arithmetic, comparison, logical operators
- **Control Flow**: if-else statements, conditional operators
- **Loops**: for loops, while loops, nested loops
- **Functions**: Function definition, parameters, return values
- **Recursion**: Base case, recursive case, recursion depth
- **List Operations**: Append, remove, sort, reverse, count
- **Dictionary Operations**: Create, add, delete, iterate
- **File I/O**: Read, write, append, string replacement
- **Object-Oriented Programming**: Classes, objects, methods
- **Modules & Packages**: Importing, organizing code

## Learning Objectives

By practicing these programs, students will learn:
1. Fundamental Python syntax and semantics
2. How to work with different data types
3. Problem-solving using conditional logic
4. Loop structures and iterations
5. Function and recursion concepts
6. Collections and data structures
7. File handling
8. Object-oriented programming basics
9. Module organization and reusability

## Requirements

- Python 3.6 or higher
- No external dependencies (uses only Python standard library)

## Notes

- All programs are designed for beginner-level learning
- Each program contains multiple methods to solve the same problem
- Interactive examples with user input are included
- Comments explain the code thoroughly
