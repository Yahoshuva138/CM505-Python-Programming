## Unit Overview

### Unit I: Foundational Programming and Control Flow (CO1 & CO2)

Learn basic Python constructs and control statements.

- **01_hello_world.py** - Your first Python program
- **02_variables_data_types.py** - Variables, data types (int, float, bool, string, list)
- **03_input_output.py** - Input/Output and formatted display
- **04_operators.py** - Arithmetic, comparison, logical, bitwise, membership, identity operators
- **05_conditional_statements.py** - if, if-else, if-elif-else statements
- **06_loop_statements.py** - for and while loops
- **07_flow_control.py** - break, continue, pass statements

**Key Concepts:**
- Variables and data types
- Operators and expressions
- Conditional logic
- Loop structures
- Program flow control

### Unit II: Functions, Modules, and Sequential Data (CO3)

Master functions and Python's extensive standard library.

- **01_functions.py** - Function definition, default args, keyword args, *args, **kwargs
- **02_lambda_functions.py** - Anonymous functions and their use cases
- **03_variable_scope.py** - Global, local, and enclosing scopes; LEGB rule
- **04_modules_packages.py** - Importing modules and using packages
- **05_string_operations.py** - String slicing, methods, formatting
- **06_arrays.py** - List operations and methods (Python's arrays)
- **07_map_reduce_filter.py** - Functional programming concepts
- **08_recursion.py** - Recursive functions with practical examples

**Key Concepts:**
- Function parameters and returns
- Lambda and functional programming
- Variable scope and namespace
- String manipulation
- List operations
- Recursion and backtracking

### Unit III: Complex Data Structures (CO4)

Work with Python's powerful collection types.

- **01_list_operations.py** - List creation, indexing, slicing, methods
- **02_list_properties.py** - Mutability, aliasing, cloning, function parameters
- **03_list_comprehension.py** - Creating lists with comprehension syntax
- **04_tuples.py** - Immutable sequences, unpacking, named tuples
- **05_tuple_comprehension.py** - Generator expressions and tuple creation
- **06_dictionaries.py** - Dict creation, methods, comprehension
- **07_sets.py** - Set operations, membership testing
- **08_collections.py** - Counter, deque, OrderedDict, defaultdict, heapq

**Key Concepts:**
- List and tuple operations
- Dictionary and set operations
- Comprehensions (list, dict, set)
- Collections module structures
- Data structure performance

### Unit IV: Advanced Programming and System Interaction (CO5)

Master OOP, file handling, and exceptions.

- **01_classes_objects.py** - Class definition, attributes, methods, properties
- **02_inheritance.py** - Single, multiple, hierarchical inheritance; MRO
- **03_overloading_overriding.py** - Method overriding, operator overloading
- **04_data_hiding_abstraction.py** - Encapsulation, private members, abstract classes
- **05_exception_handling.py** - try-except, custom exceptions, context managers
- **06_file_io.py** - Reading, writing, appending files; CSV and JSON
- **07_os_pathlib_modules.py** - Directory management, path handling

**Key Concepts:**
- Object-oriented programming
- Class hierarchies
- Data encapsulation
- Exception handling
- File I/O operations
- System interaction

## Running the Programs

### Prerequisites
- Python 3.7 or higher

### Execute a Program
```bash
python path/to/program.py
```

### Examples
```bash
# Run Unit I programs
python Unit_I_Foundational_Programming/01_hello_world.py
python Unit_I_Foundational_Programming/04_operators.py

# Run Unit IV programs
python Unit_IV_OOP_Files_Exceptions/01_classes_objects.py
python Unit_IV_OOP_Files_Exceptions/06_file_io.py
```
